# Product Overview

## Summary
- **Product**: Todo App — RESTful API for task management
- **Target Users**: General users managing personal tasks
- **Key Features**: Create, list, update, delete, filter todos by status

## Description

A simple, focused todo management API. Users can create tasks with titles and optional descriptions, mark them as completed, and filter their list by status. No authentication, no collaboration — a clean single-user task tracker.

## Target Users

| User Type | Description | Primary Need |
|---|---|---|
| General User | Anyone managing personal tasks | Track what needs to be done, mark items complete |

## Key Features

1. **Create Todo** — Add new tasks with title and optional description
2. **List Todos** — View all tasks ordered by creation date
3. **Update Todo** — Change title, description, or mark as completed
4. **Delete Todo** — Remove tasks no longer needed
5. **Filter by Status** — View only pending or only completed items
